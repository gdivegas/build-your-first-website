Introduction

Every website you visit is made using three basic languages: HTML, CSS, and (most of the time) JavaScript.

HTML is the structure of the page. CSS is the style of the page, and JavaScript handles the user interactions (ie - what happens if I click on this?

What We're Doing Today

We will construct a basic structure for our page using HTML and style the elements using CSS. We will not be using JavaScript today.

Setting Up Codepen

  1. Go to codepen.io and click "Sign Up" in the upper right-hand corner
  2. Once you're logged in, click the button that says "Create" and select "New Pen"
  3. If you'd like to use the view that Rachel showed during the demo, click "Change View" and select the first option
  4. Click the little gear icon near the CSS editor and select "Normalize" for the CSS base
  5. Click "Save & Close"

As you work through this tutorial, be sure to continue saving your work using the save button near the top.

If you get stuck or lost at any point, you can open up the examples that are embedded in the tutorial to look at the code, or you can click "Fork" to create a copy to use moving forward for your own.

Hero Section

The "hero section" of a website is the first thing that the user will see. It's usually a large banner image or pattern centered on the top of the page.

Hero Section HTML (Structure)

  1. Add a div and give it a class of "hero"
  2. Inside the div, add an h1 tag with the title of yourself, your website, or your product

Hero Section CSS (Style)

There are many different CSS properties that we'll use to style this hero section. First, click on your CSS tab and create a container where you'll style the .hero class:


.hero {

}

Inside of the two curly braces, you'll add the following properties each on a separate line with a semicolon at the end.

display: flex;
 Allows an item (in this case, the .hero div) to take up as much space as needed 
background-image: url("");
 Find an image on Unsplash that you like and put the web address for that image inside the empty quotation marks. 
background-size: cover;
 This will display your background image to cover the positioning area (the entire webpage in this example) while preserving the image proportions. 
background-position: center center;
 This will make the image centered vertically and horizontally. Try changing the first "center" to either "top" or "bottom" to see what happens. 
height: 80vh;
 vh means "viewport height" which represents how much of the page the user can see at one time. By saying 80vh, we're specifying that we want our image to cover 80% of the viewport height. 
justify-content: center;
 This specifies that we want our heading centered horizontally over the image. 
align-items: center;
 This specifies that we want our heading centered vertically over the image. 
color: white;
 Color always refers to text color, not background. I have chosen white, but feel free to experiment with other colors. 
text-shadow: black 4px 4px 20px;
 Adding a shadow to your text can sometimes make it easier to read. Experiment with changing the 4, 4, and 20 to see what they control. 
font-size: 24px;
 px means pixels. This line will change the font size for anything in our .hero div (in this case, the heading). I've chosen a font size of 24 pixels, but you can experiment with other font sizes.

Three Column Layout for Features

Next, we'll add text content below our hero section. Ultimately, we want three columns, each containing a small heading for a feature and a paragraph of text. Since the heading and text belong together, we'll group each of them with a div, then style that div.

Three Column Layout HTML (Structure)

  1. Create a div that has a class of "container"
  2. Inside of the container div, create another div that has the class of "features"
  3. Inside of features, create *three* separate divs that have the classes of feature-one, feature-two, and feature-three
  4. For each feature, add a small heading (like an h3) and a paragraph (p)

Stuck? Check out our example below:

See the Pen Three Column HTML by Rachel Warbelow (@rwarbelow) on CodePen.

Three Column Layout CSS (Style)

We'll again use display: flex; to make it so that our individual features are lined up on the page and flexing to take up space evenly.

First, we're going to make our features div display flex, in addition to giving it a property that we'll see the effect of later. In your CSS, add:


.features {
    display: flex;
    justify-content: space-between;
}

Next, let's style each of our features. Create the skeletons for each of them in your CSS like this: 


.feature-one {

}

.feature-two {

}

.feature-three {

}

Inside *each* of these containers, we'll add three properties and values:

flex-basis: 28%;
 flex-basis specifies the width of the element. In this case, we're saying that we want each of the features to be 28% of the full width of the container. 
padding: 15px;
 Padding adds space between the text and the edge of the element. By adding padding, it will make it so that the words aren't pressed right up against the edge of the next element. Feel free to experiment with the amount of padding. 
background-color: rgba(241,144,45, 0.2);
 We're giving each box a background-color. rgba is a fancy way to specify colors, where the first number, 241, represents how much red, the second number, 144, represents how much green, and the third number, 45, represents how much blue. The decimal, 0.2, represents how transparent the color is. Free free to experiment with your own numbers to produce different colors. You can also visit this website, which will allow you to pick colors from the hero image you used earlier.

Footer

We'll make a very simple footer that includes an SVG icon and a link.

Footer HTML (Structure)

Scroll to the very bottom of your HTML, and find the last two closing div tags. They look like this:


    </div>
</div>

The first tag closes our features div, and the second one closes our container div. We want our footer to be outside of the features, but still inside the container.

Between those two tags, do the following:

  1. Add a div and give it a class of "footer"
  2. Within the footer div, create another div with the class of "contact"
  3. Click on "assets", then click "icons", and click on an icon you want to represent contacting you. This will copy that code to your clipboard
  4. Paste what was copied into the contact div. It will look pretty messy, but it will work
  5. Directly under the SVG, create a link to email yourself, like this: 

<a href="mailto:rachel@girldevelopit.com">Contact Me</a>

Be sure to change the email address to your own.

Footer CSS (Style)

We want to center our icon and link, in addition to removing the default underline from our link. First, in your CSS tab on CodePen, create two containers for the style of our footer and the style of our link:


.footer {

}


a {

}

Inside of the two curly braces for footer, you'll add the following properties each on a separate line with a semicolon at the end.

margin-top: 20px;
 This will add space above the footer so that it doesn't look like it's pressed up against the three column feature layout. 
display: flex;
 Just like before, this will allow us to easily center our footer item. If we add more footer items, they'll each nicely lay up against each other in the same row. 
justify-content: space-around;
 This will make each element inside the footer have an equal amount of space around it. In our case, we only have one, so it will center the item. 
margin-bottom: 25px;
 Adding margin on bottom will make it so that the content is not up against the bottom of the screen.

Inside of the two curly braces for link (the a tag), you'll add the following properties each on a separate line with a semicolon at the end.

text-decoration: none;
 If you don't like that default underline, this is how you get rid of it. 
color: rgba(83,122,49, 1);
 I'm not a huge fan of the default blue links. This is how you'll change the color. Feel free to experiment with different colors.

Hosting Your Website

Now that you have your project built, let's upload it to the web. To do this, we'll use a service called Forge.

  1. Go to getforge.com and sign up for a free account. You'll need to confirm your email address.
  2. From your codepen, click the little button in the lower right-hand corner that says "Export" and select "Export .zip"
  3. Go back to Forge, choose a name for your first site (note, you can only change the first part -- you have to keep getforge.io on the end of it).
  4. Upload your zipped CodePen files
  5. Tada! Visit your custom link to see your site live. You can share this link with the world!
